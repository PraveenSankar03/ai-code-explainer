# AI Explainer

Django + React application that explains concepts using Gemini AI.

### Introduction

This web application is an AI-powered (Gemini) application designed to help programmers understand unfamiliar code quickly. Instead of manually analyzing complex logic, users can paste their code and receive a clear explanation generated by an AI model. The model used is Gemini-2.5-flash.

## Warning
This application uses the Gemini 2.5 Flash model on the free tier. The free tier has strict rate limits and usage quotas. Excessive or abusive usage may cause the API to temporarily rejectrequests. Please use the service responsibly.

## Setup

### Backend

cd backend
pip install -r requirements.txt
python manage.py runserver

### Frontend

cd frontend
npm install
npm start

## Environment Variables

Create:

backend/.env

Add your Gemini API key in .env:

GEMINI_API_KEY=your_api_key
